# homebrew-get

Homebrew tap for [get](https://github.com/rayinuk13/get) — download YouTube videos and audio from the terminal.

## Install

```bash
brew tap rayinuk13/get
brew install get
```

## Usage

```bash
get -mp3 <youtube-url>   # Download as MP3
get -mp4 <youtube-url>   # Download as MP4
```
